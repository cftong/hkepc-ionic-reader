HKEPC IR 是非官方、第三方的瀏覧器。本程式不會記錄任何登入名稱或密碼， 作者亦不會為本程式做成的直接或間接損失負上任何責任，用家需自行承擔風險，敬請留意！

作者利用 Web 的技術開發跨平台的程式，由於沒有得到 HKEPC 的 API 使用權限，所以只能直接讀取 hkepc.com 提供的 HTML; 但因 iOS 的 WKWebView 不允許 CORS，所以本程式必需連接 Proxy 伺服器去提取 hkepc.com 的 HTML。

Proxy 伺服器絕對不會記錄任何登入名稱或密碼。作者亦已經將 Proxy 伺服器的代碼 Open Source，如果不放心的用家可以自行架設 Proxy 伺服器！
